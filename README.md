# Run Tracker 

Run Tracker is a fitness based application that tracks all our runs and gives us statistics about our runs.<hr>

## Attributes 

1. MVVM
2. Kotlin
3. Architechtural Components 
4. Foreground Service
5. Dagger Hilt (Depenedency injection) <hr>
